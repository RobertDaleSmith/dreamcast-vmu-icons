  All icons here can be downloaded with the Dreamcast web browser.

  If you would like to download these icons with your PC and put them on your 2/4/8/16mb memory card /w PC link, then right click on the link to the icon you want and pick "Copy Shortcut". Then paste that into the address bar and change the file extension from VMI to VMS (ex.  icon.vmi to icon.vms) and press enter to download it. Then use the program called "DCI_VMI" from the tools section on the menu, to convert the VMS file to a DCI file.
